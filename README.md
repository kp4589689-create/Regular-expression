# Regular Expression (Regex) in Python

## 📌 Project Overview
This project demonstrates the practical use of **Regular Expressions (Regex)** in Python for extracting meaningful information from unstructured text data. It showcases how regex patterns can identify and retrieve different types of data such as emails, phone numbers, dates, URLs, prices, times, hashtags, and specific word patterns.

## 🎯 Objectives
- Learn the fundamentals of Regular Expressions.
- Extract structured information from raw text.
- Practice pattern matching using Python's `re` module.
- Understand how regex is used in real-world text processing.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Regular Expressions (`re` module)
- Pandas
- Matplotlib

## 📂 Features
- Extract Email Addresses
- Extract Phone Numbers
- Extract Dates
- Extract URLs
- Extract Prices
- Extract Time Formats
- Extract Hashtags
- Find words ending with **"ing"**
- Find words ending with **"tion"**
- Basic text pattern analysis

## 📚 Concepts Covered
- Regex syntax
- Character classes
- Quantifiers

- Anchors
- Groups and non-capturing groups
- Pattern matching
- Text extraction

## 📊 Applications
- Data Cleaning
- Text Mining

- Natural Language Processing (NLP)
- Information Extraction

3. Open `regular expression.ipynb`.
